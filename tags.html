O que são as tags

As tags HTML são comandos usados para orientar o navegador sobre a forma como o conteúdo deve ser exibido. Com elas, é possível aplicar formatação como itálico, inserir quebras de linha, listas com marcadores, objetos diversos e muito mais. Essas instruções fazem parte da linguagem HTML (Hypertext Markup Language), utilizada na construção de todas as páginas da web.

Tags estruturais

Exemplos de tags estruturais:

Tag de parágrafo(<p> ): Utilizada para criar parágrafos. Todo o conteúdo colocado entre as marcas de início e fim da tag será exibido como um bloco de texto comum.
Tag de título(<h1>): Serve para definir títulos ou subtítulos em uma página. Existem outras variações dessa tag, como de nível 1 a 6, que indicam diferentes níveis de importância ou hierarquia (do mais importante ao menos relevante).
Tag de cabeçalho(<header>): Representa a seção de cabeçalho da página ou de uma parte dela. Geralmente contém elementos como logotipo, menus de navegação ou títulos introdutórios.
Tag de conteúdo principal(<main>): Delimita a parte central e mais relevante do conteúdo da página — o foco principal para o usuário.
Tag de rodapé(<footer>): Indica a área inferior da página, onde normalmente se colocam informações como direitos autorais, links úteis, contatos ou redes sociais.

Tags para mídias

Exemplos de tags para mídias:

Tag de img(<img> ):Utilizada para incorporar imagens em uma página HTML.
Tag de audio(<audio>): Serve para inserir arquivos de áudio com um player embutido no navegador.
Tag de source(<source>): Define os arquivos de mídia (como áudio ou vídeo) que podem ser usados por elementos como audio ou video. Permite especificar diferentes formatos para garantir compatibilidade entre navegadores.
Tag de figure(<figure>): Agrupa conteúdos de mídia (como imagens ou áudios) junto com uma legenda opcional.
